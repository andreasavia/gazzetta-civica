<span class="keep80">2. Conigli </span>
<br/>
<span class="keep80"> </span>
<br/>
<span class="keep80">Nell'ambito della ricerca nel settore agricolo, qualora la  finalità</span>
<br/>
<span class="keep80">del progetto preveda che  gli  animali  siano  tenuti  in  condizioni</span>
<br/>
<span class="keep80">analoghe a quelle degli animali  negli  allevamenti  commerciali,  il</span>
<br/>
<span class="keep80">trattamento  degli  animali  è  conforme  almeno  alle  disposizioni</span>
<br/>
<span class="keep80">stabilite nella <a href="http://eur-lex.europa.eu/legal-content/IT/TXT/?uri=CELEX:31998L0058" target="_blank">direttiva 98/58/CE</a>. </span>

<br/>
<span class="keep80">coniglio e 60% più grande se  ne  ospita  due.  Se  si  prevede  una</span>

<br/>

<span class="keep80">più  di  ventiquattr'ore  consecutive.  I  gatti   che   manifestano</span>

<br/>
<span class="keep80">possibile trovare un compagno compatibile. È  opportuno  sorvegliare</span>
<br/>
<span class="keep80">lo stress sociale di tutti gli individui che vivono in  coppia  o  in</span>
<br/>
<span class="keep80">gruppo almeno con frequenza settimanale. Le femmine  con  piccoli  di</span>
<br/>
<span class="keep80">età inferiore a quattro settimane o che si trovano nelle ultime  due</span>
<br/>

<span class="keep80">11. Pesci </span>
<br/>
<span class="keep80"> </span>
<br/>
<span class="keep80">11.1. Fornitura e qualità dell'acqua </span>
<br/>
<span class="keep80"> </span>
<br/>
<span class="keep80">È necessario fornire continuamente acqua a sufficienza e di qualità</span>
<br/>
<span class="keep80">adeguata. Il flusso d'acqua nei sistemi di ricircolo o il  filtraggio</span>
<br/>
<span class="keep80">all'interno delle vasche deve essere sufficiente e  garantire  che  i</span>
<br/>
<span class="keep80">parametri  di  qualità  dell'acqua   siano   mantenuti   a   livelli</span>
<br/>
<span class="keep80">soddisfacenti, in  funzione  delle  caratteristiche  del  sistema  di</span>
<br/>
<span class="keep80">allevamento, delle specie e delle esigenze della fase di sviluppo. Se</span>
<br/>
<span class="keep80">necessario, l'acqua deve essere filtrata o trattata per eliminare  le</span>
<br/>
<span class="keep80">sostanze che possano  nuocere  ai  pesci.  I  parametri  di  qualità</span>
<br/>
<span class="keep80">dell'acqua devono rimanere sempre entro  intervalli  accettabili  che</span>
<br/>
<span class="keep80">permettano lo svolgimento  dell'attività  normale  e  sostengano  la</span>
<br/>
<span class="keep80">fisiologia di una specie e di una fase di  sviluppo  determinate.  Il</span>
<br/>
<span class="keep80">flusso d'acqua deve permettere ai pesci di nuotare correttamente e di</span>
<br/>
<span class="keep80">mantenere un comportamento normale. Deve  essere  previsto  il  tempo</span>
<br/>
<span class="keep80">necessario  per  l'acclimatazione  e  l'adattamento  dei   pesci   ai</span>
<br/>
<span class="keep80">cambiamenti nella qualità  dell'acqua.  Si  devono  adottare  misure</span>
<br/>
<span class="keep80">opportune per  ridurre  al  minimo  variazioni  improvvise  dei  vari</span>
<br/>
<span class="keep80">parametri  che  influiscono  sulla   qualità   dell'acqua.   Occorre</span>
<br/>
<span class="keep80">monitorare il flusso e il livello dell'acqua e  garantire  che  siano</span>
<br/>
<span class="keep80">adeguati. </span>
<br/>
<span class="keep80"> </span>
<br/>
<span class="keep80">11.2.  Ossigeno,  composti  azotati,  biossido  di  carbonio,  pH   e</span>
<br/>
<span class="keep80">salinità </span>
<br/>
<span class="keep80"> </span>
<br/>
<span class="keep80">La concentrazione di ossigeno deve  essere  appropriata  alle  specie</span>
<br/>
<span class="keep80">interessate e al contesto  nel  quale  vivono.  Se  necessario,  deve</span>
<br/>
<span class="keep80">essere fornita un'aerazione supplementare dell'acqua della vasca,  in</span>
<br/>
<span class="keep80">funzione del sistema di allevamento. Le concentrazioni di biossido di</span>
<br/>
<span class="keep80">carbonio e di composti azotati, in particolare ammoniaca,  nitriti  e</span>
<br/>
<span class="keep80">nitrati, devono essere mantenute al di sotto dei livelli  nocivi.  La</span>
<br/>
<span class="keep80">qualità dell'acqua deve essere monitorata utilizzando un  calendario</span>
<br/>
<span class="keep80">di analisi definito con  una  frequenza  sufficiente  a  rilevare  le</span>
<br/>
<span class="keep80">variazioni di questi  parametri  critici  e  devono  essere  adottate</span>
<br/>
<span class="keep80">misure per mitigare tali variazioni. Il pH deve essere adattato  alle</span>
<br/>
<span class="keep80">specie e monitorato per mantenerlo  il  più  possibile  stabile.  La</span>
<br/>
<span class="keep80">salinità deve essere adattata alle esigenze della  specie  ittica  e</span>
<br/>
<span class="keep80">alla fase di sviluppo degli  animali.  Le  modifiche  alla  salinità</span>
<br/>
<span class="keep80">devono essere introdotte gradualmente. </span>
<br/>
<span class="keep80"> </span>
<br/>
<span class="keep80">11.3. Temperatura e illuminazione </span>
<br/>
<span class="keep80"> </span>
<br/>
<span class="keep80">La temperatura va mantenuta entro l'intervallo ottimale per la specie</span>
<br/>
<span class="keep80">interessata e lo stadio di sviluppo e va mantenuta il più  possibile</span>
<br/>
<span class="keep80">stabile. Le  modifiche  alla  temperatura  devono  essere  introdotte</span>
<br/>
<span class="keep80">gradualmente. Occorre prevedere un fotoperiodo adeguato ai pesci. </span>
<br/>
<span class="keep80"> </span>
<br/>
<span class="keep80">11.4. Densità di popolamento e complessità ambientale </span>
<br/>
<span class="keep80"> </span>
<br/>
<span class="keep80">La densità di popolamento  deve  essere  determinata  in  base  alle</span>
<br/>
<span class="keep80">esigenze complessive dei  pesci  rispetto  a  condizioni  ambientali,</span>
<br/>
<span class="keep80">salute e benessere. I pesci devono avere  a  disposizione  un  volume</span>
<br/>
<span class="keep80">d'acqua sufficiente per poter nuotare normalmente tenuto conto  della</span>
<br/>
<span class="keep80">dimensione,  dell'età,  dello  stato  di  salute  e  dei  metodi  di</span>
<br/>
<span class="keep80">nutrimento.  Deve  essere  previsto   per   i   pesci   un   adeguato</span>
<br/>
<span class="keep80">arricchimento ambientale, ad esempio nascondigli o substrati, a  meno</span>
<br/>
<span class="keep80">che  in  base  ai  tratti  comportamentali  dei  pesci  non   risulti</span>
<br/>
<span class="keep80">necessario. </span>
<br/>
<span class="keep80"> </span>
<br/>
<span class="keep80">11.5. Alimentazione e manipolazione </span>
<br/>
<span class="keep80"> </span>
<br/>
<span class="keep80">L'alimentazione deve  corrispondere  alle  esigenze  dei  pesci,  che</span>
<br/>
<span class="keep80">devono essere nutriti ad una velocità e ad una  frequenza  adeguate.</span>
<br/>
<span class="keep80">Occorre  prestare  particolare  attenzione  all'alimentazione   delle</span>
<br/>
<span class="keep80">larve, quando si passi da alimenti naturali ad alimenti  artificiali.</span>
<br/>
<span class="keep80">Se necessaria per motivi non procedurali (ad esempio  il  trasporto),</span>
<br/>
<span class="keep80">la sospensione dell'alimentazione deve essere il più breve possibile</span>
<br/>
<span class="keep80">e deve tenere conto della dimensione dei pesci  e  della  temperatura</span>
<br/>
<span class="keep80">dell'acqua. </span>
<br/>
<span class="keep80">Laddove possibile, i pesci devono essere manipolati senza  rimuoverli</span>
<br/>
<span class="keep80">dall'acqua. La manipolazione dei pesci sia dentro l'acqua  che  fuori</span>
<br/>
<span class="keep80">deve essere ridotta al minimo e le attrezzature  a  diretto  contatto</span>
<br/>
<span class="keep80">con i pesci devono essere umidificate.  I  pesci  non  devono  essere</span>
<br/>
<span class="keep80">manipolati  a  una  temperatura  che   si   avvicini   agli   estremi</span>
<br/>
<span class="keep80">dell'intervallo di temperature dell'acqua che essi sono in  grado  di</span>
<br/>
<span class="keep80">tollerare. </span>
<br/>
<span class="keep80"> </span>

<span class="keep80">11.6.2. Illuminazione </span>

<span class="keep80"> </span>

<span class="keep80">11.6.3. Densità di popolamento e complessità ambientale </span>
<br/>
<span class="keep80"> </span>
<br/>
<span class="keep80">Per i pesci zebra adulti non devono essere utilizzati volumi  d'acqua</span>
<br/>
<span class="keep80">inferiori a 1 litro. Le densità di popolamento non  devono  superare</span>
<br/>
<span class="keep80">10 pesci adulti/litro. Le dimensioni e la forma  della  vasca  devono</span>
<br/>
<span class="keep80">consentire ai pesci di mantenere il comportamento e le  attività  di</span>
<br/>
<span class="keep80">nuoto naturali. </span>
<br/>
<span class="keep80"> </span>
<br/>
<span class="keep80">Deve essere evitato un alloggiamento singolo prolungato. </span>
<br/>
<span class="keep80"> </span>

<span class="keep80">12. Cefalopodi </span>
<br/>
<span class="keep80"> </span>
<br/>
<span class="keep80">12.1. Fornitura e qualità dell'acqua </span>
<br/>
<span class="keep80"> </span>
<br/>
<span class="keep80">È necessario fornire continuamente acqua a sufficienza e di qualità</span>
<br/>
<span class="keep80">adeguata. </span>
<br/>
<span class="keep80"> </span>
<br/>
<span class="keep80">La progettazione della vasca e il flusso dell'acqua devono soddisfare</span>
<br/>
<span class="keep80">le  esigenze  dell'animale,  compresa  un'adeguata  ossigenazione  in</span>
<br/>
<span class="keep80">funzione della dimensione,  della  fase  di  vita  e  delle  esigenze</span>
<br/>
<span class="keep80">comportamentali. La temperatura, la salinità, il pH e i livelli  dei</span>
<br/>
<span class="keep80">composti azotati dell'acqua  devono  essere  adeguati  alle  esigenze</span>
<br/>
<span class="keep80">delle specie e  delle  forme  di  vita.  Le  fughe  e  l'introduzione</span>
<br/>
<span class="keep80">accidentale di elementi estranei devono essere  evitate  utilizzando,</span>
<br/>
<span class="keep80">se necessario, coperture. </span>
<br/>
<span class="keep80"> </span>
<br/>
<span class="keep80">Deve essere previsto  il  tempo  necessario  per  l'acclimatazione  e</span>
<br/>
<span class="keep80">l'adattamento  dei   cefalopodi   ai   cambiamenti   nella   qualità</span>
<br/>
<span class="keep80">dell'acqua. </span>
<br/>
<span class="keep80"> </span>
<br/>
<span class="keep80">12.2. Illuminazione </span>
<br/>
<span class="keep80"> </span>
<br/>
<span class="keep80">L'intensità luminosa e il fotoperiodo devono soddisfare i  requisiti</span>
<br/>
<span class="keep80">della specie. </span>
<br/>
<span class="keep80"> </span>
<br/>
<span class="keep80">12.3. Alimentazione </span>
<br/>
<span class="keep80"> </span>
<br/>
<span class="keep80">Ai cefalopodi deve essere fornito un regime di alimentazione adeguato</span>
<br/>
<span class="keep80">alla specie, alla fase di sviluppo e alle esigenze comportamentali. </span>
<br/>
<span class="keep80"> </span>
<br/>
<span class="keep80">12.4. Arricchimento e manipolazione </span>
<br/>
<span class="keep80"> </span>
<br/>
<span class="keep80">I cefalopodi devono ricevere una quantità adeguata e sufficiente  di</span>
<br/>
<span class="keep80">stimoli fisici, cognitivi e sensoriali per consentire un'ampia  gamma</span>
<br/>
<span class="keep80">di comportamenti propri della specie. Le condizioni di  alloggiamento</span>
<br/>
<span class="keep80">devono tenere conto delle esigenze sociali proprie della  specie  (ad</span>
<br/>
<span class="keep80">esempio, abitudini di vita solitaria o di gruppo). Laddove  opportuno</span>
<br/>
<span class="keep80">per la specie, devono essere forniti rifugi o tane. </span>
<br/>
<span class="keep80"> </span>
<br/>
<span class="keep80">Laddove  possibile,  i  cefalopodi  devono  essere  manipolati  senza</span>
<br/>
<span class="keep80">rimuoverli dall'acqua. La manipolazione  dei  cefalopodi  sia  dentro</span>
<br/>
<span class="keep80">l'acqua che fuori deve essere ridotta al minimo e le  attrezzature  a</span>
<br/>
